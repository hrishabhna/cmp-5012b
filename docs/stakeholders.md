# Stakeholder List

| Stakeholder        | Role / Type             | Needs / Concerns                          |
|--------------------|-------------------------|-------------------------------------------|
| General public     | Primary end user        | There should be easy navigation so that anyone from any age would be able to use it. It should also have the ability to be able to keep record of previous logs so that the customer would be able to see progress.|
| Admin              | System administrator    | The program itself should be easily maintainable. Data needs to be stored securely.|
| Government         | External                | As the data is personal and confidential the government would need the data to be secured safely as well as checking to see if the companies are following the correct procedures in terms of data handling. |
