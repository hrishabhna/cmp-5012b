# User Stories

Use the format:

## Story List

1. As a user, I want to edit my previous health enteries so I can correct mistakes.  # Important
2. As a user I want a generated weekly report so i can anaylse data trends.  # Important
3. As a user i want to receive reminders to log my health data so that i do not forget.
4. I want to be able to change and update my profile information so my account details remain accurate  # Important
5. As a full time worker, I want to organise my schedule so that I can reserve time to utilise my health tracker
6. As a retired person, I want to be able to understand how to use the health tracker with little to no reading on how it works so that I can understand how it works easily
7. As a person that goes on runs, I want my health tracker to tell me the most effective ways running can improve my weight so that I can implement these tips into my sessions 
8. As a networker, I want my health tracker to show me other people's data such as friends so that I can see what's most effective for the people around me at improve their health
