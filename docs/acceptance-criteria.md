## Story: [ID / Title]

- Given that users will want reports of their health
  When week has past
  Then health tracker should automatically create a weekly report comprised of the user's health data of that week and show the user in that report what the data indicates about the user's health

- Given the user wants to update their profile information
  When their current profile information is inaccurate
  Then the health tracker should give the user the option to change their profile information at anytime
